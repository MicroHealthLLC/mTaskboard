What is this?

mTask Board is a simple and beautiful Chrome Apps for creating and managing todo lists locally.

How does it work?

It's built in HTML5 and Javascript, using local storage to save your todos in the browser instead of in a web server, which makes the app availible offline aswell.

Why doesn't it work?

Are you using Internet Explorer? No? Try pressing here. This should reset the mTask Board, and make it functional again. However, all your current todos will be lost. Sorry.

Who created this?

Todo App was originally created created by Filip Stefansson at Pixby Media. It was modified by MicroHealth. for internal use as a Chrome App. The source is of course open...you've downloaded it if you downloaded this app. You can get the original source a Github.

Whats next?

Google Task synchronization. Simply put, we use Google Apps for our backbone and love it.

Where can I get more help for this?

Remember, this was created internally for MicroHealth Employees so support is limited. You can always contact us here.
